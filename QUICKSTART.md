# 快速开始

## 项目结构

```
mini/
├── app.js                 # 小程序入口文件
├── app.json              # 全局配置
├── app.wxss              # 全局样式
├── pages/                # 页面目录
│   ├── splash/          # 启动页
│   ├── index/           # 首页（情绪输入）
│   ├── analysis/        # 情绪解析页
│   ├── action/          # 微行动建议页
│   └── history/         # 情绪记录页
└── project.config.json   # 项目配置
```

## 使用步骤

1. **打开微信开发者工具**
   - 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)

2. **导入项目**
   - 打开微信开发者工具
   - 选择"导入项目"
   - 选择项目目录：`/Users/Wangjian/Desktop/mini`
   - AppID 可以选择"测试号"或使用自己的 AppID

3. **编译运行**
   - 点击"编译"按钮
   - 在模拟器中查看效果

## 功能流程

1. **启动页** → 点击"开始说说看"
2. **首页** → 输入情绪文字 → 点击"我说完了"
3. **情绪解析页** → 查看情绪标签和解释 → 点击"我想知道接下来怎么做"
4. **微行动建议页** → 查看建议 → 选择"我知道了"或"记录下来"
5. **情绪记录页**（可选）→ 查看历史记录和趋势

## 设计特点

- ✅ 极简设计，3步完成主流程
- ✅ 暖色调，大留白，大字号
- ✅ 不评判、不说教的文案
- ✅ 无弹窗、无强制注册
- ✅ 情绪分析基于关键词匹配（MVP版本）

## 注意事项

- 当前版本数据存储在内存中，刷新后会丢失
- 情绪分析使用简单的关键词匹配，生产环境建议接入专业NLP服务
- 建议在真机上测试以获得最佳体验
